Conceito Matemático: Detecção de Diferenças em Imagens

O objetivo é detectar onde uma imagem foi alterada em relação à original.

📐 Fórmula:

    diff(x, y) = |R1 - R2| + |G1 - G2| + |B1 - B2|
    Se diff(x, y) > limiar → existe uma diferença significativa nesse pixel

Cada pixel da imagem original é comparado com o correspondente na imagem modificada. Se a soma das diferenças dos canais de cor (vermelho, verde, azul) for maior que um valor pré-definido (limiar), consideramos aquele ponto como uma "mudança".

🧠 Aplicação

  • Comparação é feita pixel a pixel.
  • Coordenadas com diferenças são registradas.
  • Imagem gerada destaca esses pontos em vermelho.
